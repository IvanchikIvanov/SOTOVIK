# ZABERG Website UI Kit

Interactive click-through prototype of the ZABERG premium electronics e-commerce website.

## Screens

1. **Homepage** — Hero banner with texture, category grid, featured products
2. **Catalog / PLP** — Filter sidebar + product grid with chips/filters
3. **Product Detail / PDP** — Specs, images, add to cart
4. **Cart** — Cart drawer / mini cart

## Files

- `index.html` — Main interactive prototype (all screens)
- `Header.jsx` — Site header with logo, search, nav, cart icon
- `ProductCard.jsx` — Product card component
- `CategoryGrid.jsx` — Category tiles grid
- `FilterPanel.jsx` — Filter sidebar with chips
- `ProductList.jsx` — Product listing page layout

## Design Notes

- Dark base: `#0a0a0a` background
- Display font: Unbounded (Google Fonts substitute for brand font)
- Body: Inter
- Primary accent: `#d93caf` (magenta)
- Textures: stripe PNGs used as hero/section backgrounds with dark scrim
- No codebase was available — UI is designed from brand description + textures
